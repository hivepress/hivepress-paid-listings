== Description ==

HivePress Paid Listings is an extension for [HivePress](https://wordpress.org/plugins/hivepress/) plugin. It allows you to charge users for adding, featuring and renewing listings.

[Demo](https://demo.hivepress.io/) | [Docs](https://hivepress.io/docs/extensions/paid-listings/?utm_medium=referral&utm_source=wordpress.org) | [Support](https://hivepress.io/support/forum/extensions/paid-listings/?utm_medium=referral&utm_source=wordpress.org)
